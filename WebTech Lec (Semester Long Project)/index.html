Glossary

C

Cascading Style Sheets : a language used to specify the presentation aspects (i.e. layout and formatting of structurally marked up (e.g. HTML, XML, XHTML, SVG, etc.) document).

D

Defense Advanced Research Projects Agency(DARPA): an agency of the United States Department of Defense that develops technologies for the military.

DOCTYPES : Document Type Declaration

E

F

G

Gopher Protocol : Document Type Declaration

H

Hyper Text Markup Language(HTML) : a language used to markup documents (i.e. web pages) in the WWW.

Hyper Text Transfer Protocol(HTTP) : an application layer used primarily to retrieve hypertext (on hypermedia) documents and resources on the WWW.

Hyper Text Transfer Protocol Secure(HTTPS) : the secure version of HTTP.

I

Internet : a global information system of interconnected computer networks that is logically linked together by a globally unique address space based on the IP or its subsequent extensions

Internet Engineering Task Force(IETF) : a huge international community of network engineers, designers, researchers and a like concerning to the evolution of the Internet architecutre.

Internet Protocol(IP) : a set of rules that contols over the Internet the format of data.

SSL: Secure Socket Layer

T

Tim-Berners Lee: the father of the WWW.

TCP: Transmission Control Protocol

TLS: Transport Layer Security

U

Usenet : a worldwide distributed discussion system available on computers.

V

Vinton Cerf : the father of the Internet

W

Wide Area Information Servers(WAIS) : an Internet system in which specialized subject databases are created at multiple server locations, kept track of by a directory of servers at one location, and made accessible for searching by users with WAIS client programs.

W3C : World Wide Web Consortium

World Wide Web(WWW) : a global information medium which users can read and write via computers connected to the Internet.
